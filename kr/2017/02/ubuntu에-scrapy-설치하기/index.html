Ubuntu에 Scrapy 설치하기

Posted on February 7, 2017

본 문서는 Scrapy 공식 문서의 Installing Scrapy를 참고 했습니다. 해당 문서에 따르면 Scrapy는 Ubuntu 12.04 이상을 지원하고 있습니다. 또한 Ubuntu에서 자체 지원하는 python-scrapy는 Scrapy의 최신 버전과는 성능등의 측면에서 사용하지 말것을 권고하고 있습니다. 아래의 작업은 Ubuntu 16.04.1 LTS 에서 실행 됐습니다.

관련 패키지 설치

Scrapy를 설치하기 전에 실행하기 위해 필요한 연관 패키지들을 아래의 명령으로 설치합니다.

sudo apt-get install python-dev python-pip libxml2-dev libxslt1-dev zlib1g-dev libffi-dev libssl-dev

Scrapy를 Python3로 활용하고 싶을 경우에는 아래의 명령으로 필요한 패키지를 먼저 설치합니다.

sudo apt-get install python3 python3-dev

Scrapy 설치

Installing Scrapy 문서에서는 Scrapy를 Virtualenv에서 설치 활용할 것을 권장하고 있습니다. Virtualenv에 대한 상세한 활용 방법은 사용자 가이드에 자세한 설명이 있습니다.

sudo pip install virtualenv

설치가 완료되면 Virtualenv를 이용할 환경을 정의해주고 해당 폴더로 이동합니다. 아래 명령어를 기준으로 -p python3는 실행환경을 Python3로 정의하고 폴더는 scrapy로 지정됩니다.

virtualenv -p python3 scrapy

scrapy 폴더로 이동해서 source bin/activate 명령어를 수행하면 Virtualenv 실행이 준비되고 Scrapy 설치를 위한 모든 준비 과정이 마무리 됩니다.

pip install scrapy

이제 Scrapy 실행을 위한 모든 준비가 끝났습니다.
