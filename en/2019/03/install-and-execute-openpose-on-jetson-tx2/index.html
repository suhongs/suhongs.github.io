Install and execute Openpose on Jetson Tx2

Jetson Tx2 Openpose camera

Install Jetpack 3.3 on Jetson Tx2

There's a lot of resources to do. I used this link. The post used Virtual box and virtual box extension.

Install OpenPose

To use jetson tx2 onboard camera, it is necessary to install opencv3.4. First, I removed prior version of opencv according to this article.

$ sudo apt-get purge libopencv*
$ sudo apt-get purge python-numpy

I used shell script to install opencv 3.4 version from this repository.

$ cd buildOpenCVTX2
$ ./buildOpenCV.sh

After installing opencv 3.4, I installed caffe and openpose using shell script from official repository.

$ cd openpose
$ ./scripts/ubuntu/install_caffe_and_openpose_JetsonTX2_JetPack3.3.sh

To use on board camera, I modified source code and recompile openpose. This is not recommended if you do not have enough knowledge about C++. You are able to check the code from this link.

Compile OpenPose using ccmake

$ sudo apt-get install cmake-curses-gui
$ mkdir build
$ cd build
$ ccmake ..
$ sudo make install -j`nproc`
